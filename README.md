# 📊 Big Data Analytics for Retail Demand

## 📌 Overview

This project analyzes retail demand using Python, Pandas, and SQLite. It demonstrates the ETL (Extract, Transform, Load) process and simulates Big Data processing using groupby operations.

## 🎯 Objective

To analyze retail sales data and identify patterns in demand across categories, countries, customer segments, and time.

## 🛠 Tools & Technologies

* Python
* Pandas
* SQLite
* Matplotlib
* Jupyter Notebook

## 🔄 Process

* Data loading and preprocessing
* Data cleaning and transformation
* Aggregation using groupby (MapReduce simulation)
* Data storage using SQLite
* SQL queries for analysis
* Visualization of trends

## 📈 Key Insights

* Sales vary across product categories and customer segments
* Demand trends change over time
* Average-based analysis provides better understanding due to uneven data distribution

## 📌 Conclusion

This project demonstrates how Big Data concepts like ETL and MapReduce can be applied to smaller datasets to extract meaningful insights and support data-driven decision making.
